# braga-weather-dashboard

🌧️ Dashboard Meteorológico — Braga, Fevereiro 2026

📌 Sobre o Projeto

Dashboard interativo de análise meteorológica da cidade de Braga, Portugal, desenvolvido inteiramente em HTML, CSS e JavaScript. O projeto apresenta dados climáticos de fevereiro de 2026 com visualizações dinâmicas, animações e calendário interativo.


💡 Não requer instalação — basta abrir o arquivo .html no navegador!

📊 O que o Dashboard apresenta

🌡️ Temperatura média mensal: 12,8°C (máx.) / 5°C (mín. noturna)
🌧️ Precipitação total: 89 mm no mês
📅 Calendário meteorológico dia a dia com tooltips interativos
📈 Gráfico de linha — evolução de temperatura máx./mín. ao longo do mês
📊 Gráfico de barras — dias de chuva vs dias de sol por semana
🍩 Gráfico de composição — distribuição dos tipos de dia (sol, chuva, nublado)
🔵 Gráfico de dispersão — correlação entre temperatura e horas de sol
☁️ Animações de fundo — nuvens e chuva animadas em CSS

🛠️ Tecnologias Utilizadas


HTML5 — estrutura e semântica
CSS3 — layout com Grid/Flexbox, animações, design responsivo
JavaScript — interatividade e lógica dos filtros
Chart.js — biblioteca de gráficos interativos
Google Fonts — tipografia (Playfair Display + DM Sans)

🚀 Como Visualizar

Baixa o arquivo braga_weather_dashboard.html
Abre no navegador (Chrome, Firefox, Edge)
Pronto — sem instalação necessária!

Ou acesse diretamente pelo GitHub Pages (se habilitado no repositório).
